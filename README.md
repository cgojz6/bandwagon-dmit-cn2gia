## 先搞清楚：CN2 GIA是什么

**CN2 GIA（Global Internet Access）**是中国电信旗下的高端骨干网线路，属于CN2系列中最顶级的一档。它与普通国际线路最大的区别在于：数据包从海外到国内全程走专用高速通道，不经过拥挤的公共骨干节点。

简单翻译一下：晚高峰别人卡成PPT，你该跑多快还是多快。

对国内用户来说，CN2 GIA的实际意义是——延迟低、丢包少、晚高峰稳定。这正是搬瓦工和DMIT共同的核心卖点。
